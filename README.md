# DIP_2022

## Setup Environments
- Create virtual environment
```bash
python3.8 -m venv .venv  
```

- Activate virtual environment
```bash
source .venv/bin/activate
```

- Install packages
```bash
pip install -U pip
pip install -r requirements.txt
```
